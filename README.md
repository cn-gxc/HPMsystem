# 学生家务劳动实践管理系统

这是一个基于网页的学生家务劳动管理系统，支持学生上传家务照片，班主任审核计分。

## 功能特点
- 学生上传每周家务照片
- 班主任审核系统
- 积分管理系统
- 使用Supabase作为后端数据库
- 响应式设计，支持移动设备

## 使用方法
1. 访问网站
2. 配置Supabase数据库
3. 初始化数据库表
4. 使用默认账号登录测试
   - 学生：student123 / 123456
   - 教师：teacher123 / 123456

## 技术栈
- HTML5, CSS3, JavaScript
- Supabase (PostgreSQL数据库)
- GitHub Pages (托管)
